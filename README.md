# Personal Portfolio Website

## Description

A personal portfolio website built using HTML, CSS, and JavaScript. The website showcases my skills, projects, education, and contact information in a clean and responsive design.

## Features

* Responsive design
* About Me section
* Skills section
* Projects showcase
* Contact section
* Modern user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in a web browser.
